**功能：**
- 🎭 **6 种场景人格**：工作 / 旅行 / 亲密 / 危机 / 社交 / 内容消费
- 👤 **人物记忆系统**：存储身边人的特征（不吃辣、急性子、讨厌被说教），对话时自动注入
- 🟢🟡🔴 **红绿灯上下文控制**：始终注入 / 关联时注入 / 仅存储
- 🧑 **用户背景面板**：年龄、职业、偏好——AI 据此个性化所有输出
- 🌐 **中英文切换**：一键切换全部 UI 和 AI 输出语言
- 🔌 **多供应商支持**：DeepSeek / OpenAI / 自定义兼容接口

## 不是什么

- ❌ 不是一个聊天机器人
- ❌ 不是一个需要打开才能用的 App
- ❌ 不是一个只会回答问题的 AI

## 是什么

一个**环境感知、人格自适应、持续在线、双向代理**的 AI 共生层。

它像你的第二层意识——

- **不等待指令**。从环境中推断你现在需要什么。
- **不需要打字**。读取你的注意力焦点和真实意图。
- **不只有文字**。向大脑传递直觉般的微弱信号。
- **不止对你说话**。替你向外部系统沟通——在你授权的前提下。

## 🔑 关键设计原则

| # | 原则 | 说明 |
|---|------|------|
| 1 | 主动感知 | AI 从环境推断场景，不等用户指令 |
| 2 | 意图与行为分离 | 人表面上做 A，真实想的是 B——AI 理解 B |
| 3 | 双向信息流 | AI 对人输出 + 对外界代理输出 |
| 4 | 细粒度同意 | 不是一次性授权——每场景、每数据、每次输出可控 |
| 5 | 人格随场景 | 同一个 AI，工作是分析师，旅行是向导，深夜是沉默陪伴 |

## 🏷 为什么叫 Symbiote

生物学概念：两种生物共同生活、互相受益。这个 AI 与你共生——你是它的感知来源和决策终点，它是你的信息过滤器和意图放大器。它不是你，但它懂你。
